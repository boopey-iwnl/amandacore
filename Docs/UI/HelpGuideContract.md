# Help Guide Contract

The Help / Guide panel is a first-party AmandaCore UI panel. It is built into the client and uses AmandaCore-original text.

## Entry Points

- Game Menu: Help / Guide.
- Micro menu: Help.
- Pre-world screens: Help.
- Chat: `/help`.
- Tutorial hint: Open Help.

## Categories

- Getting Started
- Movement and Camera
- Combat
- Quests
- Inventory and Equipment
- Spellbook and Action Bars
- Map and Objectives
- Chat and Social
- Settings and Keybinds
- Known Alpha Limitations

## Rules

- Help text must describe actual AmandaCore features and current alpha limitations.
- Search/filter is local UI filtering only.
- Help does not load external modules, external packs, scripts, Lua, addons, or plugin content.
- Help must close through the panel close button and standard Escape panel ordering.
- Help must remain readable at default UI scale.
